HTML5 Drag and Drop

Scenario 1: Enabling draggable elements

Back to scenarios menu

Description

Set the draggable attribute to 'true' to let elements be dragged on the page, or to 'false' to block elements from being dragged.

Code

<pre>
    <!-- Enable draggable <div> elements, and block images and links -->
    <div draggable="true">This div is draggable </div>
    <div draggable="true">
        <p>
            This div is draggable and contains some <a href="">links</a>
            and images which are not draggable.
        </p>
        <img draggable="false" alt="smiley flower" src="images/smiley.png" />
        <p>
            <a draggable="false" href="">another link</a>
        </p>
    </div>
</pre>

Result

Try dragging these elements to see what happens.
