HTML5 Drag and Drop

Scenario 3: Acting on the area being dragged over

Back to scenarios menu

Description

Elements which are being dragged over can act on dragenter, dragleave, dragover, or drop events.

Note that most elements are not automatically drop targets. Text input elements and contenteditable elements can receive the drop event by default but other elements will not. In order to make an element a drop target you must cancel the dragenter and dragover events, then handle the drop event.

Code

<!-- Change color randomly when an event occurs -->
<div ondragenter="backgroundChange(this);">
    This element changes background color on the dragenter event.
</div>
<div ondragover="backgroundChange(this);">
    This element changes background color on the dragover event.
</div>
<div ondragleave="backgroundChange(this);">
    This element changes background color on the dragleave event.
</div>
<div ondragover="return false;" ondrop="backgroundChange(this);">
    This element changes background color on the drop event. 
</div>

<script>
    var colors = ["red", "green", "blue", "orange", "purple", "yellow", "lime"];
    function chooseColor() {
        var num = Math.floor(Math.random()*colors.length);
        return colors[num];
    }

    function backgroundChange(elt) {       
        elt.style.backgroundColor = chooseColor();
    }
</script>
