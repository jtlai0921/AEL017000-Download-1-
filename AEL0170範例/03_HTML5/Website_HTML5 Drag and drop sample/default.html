HTML5 Drag and Drop

Scenario 1: Enabling draggable elements

Set the draggable attribute to 'true' to let elements be dragged on the page, or to 'false' to block elements from being dragged.

Scenario 2: Acting on the element being dragged

Dragged elements can act on three different events: dragstart, drag, and dragend.

Scenario 3: Acting on the area being dragged over

Elements which are being dragged over can act on dragenter, dragleave, dragover, or drop events.

Scenario 4: Putting it all together

Use the dataTransfer object from drag events to set the data to be transferred between the item being dragged, and the element being dropped on.
