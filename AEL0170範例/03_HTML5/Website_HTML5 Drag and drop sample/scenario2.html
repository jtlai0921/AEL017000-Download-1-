Scenario 2: Acting on the element being dragged

Back to scenarios menu

Description

Dragged elements can act on three different events: dragstart, drag, and dragend.

Code

<!-- Change color randomly when an event occurs -->
<div draggable="true" ondragstart="backgroundChange(this);">
    This element changes background color on the dragstart event.
</div>
<div draggable="true" ondrag="backgroundChange(this);">
    This element changes background color on the drag event.
</div>
<div draggable="true" ondragend="backgroundChange(this);">
    This element changes background color on the dragend event.
</div>

<script>
    var colors = ["red", "green", "blue", "orange", "purple", "yellow", "lime"];

    function chooseColor() {
        var num = Math.floor(Math.random()*colors.length);
        return colors[num];
    }

    function backgroundChange(elt) {
        elt.style.backgroundColor = chooseColor();
    }
</script>
